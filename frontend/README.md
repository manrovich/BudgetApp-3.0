# React + TypeScript + Vite

## Хочу поднять проект локально
Перед тем как поднимать проект требуется в корневой папке проекта установить зависимости
```bash
npm i
```
[что делать если нет npm](https://nodejs.org/en/download)

и поднимаем проект локально в dev режиме, а больше нам и не надо
```bash
npm run dev
```
